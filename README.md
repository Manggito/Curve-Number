# CN Composite Calculator

Web statis untuk menghitung CN komposit berbasis ESRI Land Cover, FAO Soil, HSG, dan metode USDA SCS-CN.

## Isi file

- `index.html`
- `style.css`
- `script.js`
- `.nojekyll`

## Cara membuka di komputer

Cukup buka file `index.html` di browser.

## Cara upload ke GitHub Pages

1. Buat repository baru di GitHub.
2. Upload semua file di folder ini ke repository.
3. Buka menu `Settings`.
4. Pilih `Pages`.
5. Pada bagian `Build and deployment`, pilih `Deploy from a branch`.
6. Pilih branch `main` dan folder `/root`.
7. Klik `Save`.
8. Tunggu proses deploy selesai.
9. Web akan tersedia pada alamat seperti:

```text
https://username.github.io/nama-repository/
```

## Catatan

Project ini sudah diubah dari React JSX menjadi JavaScript murni. Tidak perlu `npm install`, Vite, React, atau proses build.
