# CN Composite Calculator

Web statis untuk menghitung CN komposit berdasarkan ESRI Land Cover dan FAO Soil HSG.

## Isi folder

- `index.html` sebagai halaman utama.
- `style.css` sebagai desain tampilan.
- `script.js` sebagai logika JavaScript.

## Cara membuka di laptop

Klik dua kali file `index.html`, atau buka dengan browser.

## Cara upload ke GitHub Pages

1. Buat repository baru di GitHub.
2. Upload semua file dari folder ini ke repository:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Masuk ke `Settings`.
4. Pilih `Pages`.
5. Pada bagian `Build and deployment`, pilih `Deploy from a branch`.
6. Pilih branch `main` dan folder `/root`.
7. Simpan.
8. Buka alamat GitHub Pages yang muncul.

## Catatan

Project ini tidak membutuhkan React, Vite, npm, atau proses build.
